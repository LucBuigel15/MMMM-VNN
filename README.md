## Installatie

**1. Clone het project**
```bash
git clone git@github.com:LucBuigel15/MMMM-VNN.git
cd MMMM-VNN
```

**2. Installeer de dependencies**
```bash
npm install
```

**3. Stel de omgevingsvariabelen in**
```bash
cp .env.example .env
# Vul je Supabase URL en API-sleutel in in het .env bestand
```

**4. Start de ontwikkelserver**
```bash
npm run dev
```
